# Coca-Cola Financial Performance Dashboard (FY2009–FY2018)

## Overview
A financial analysis project covering 10 years of Coca-Cola's Profit & Loss, 
Balance Sheet, and Cash Flow statements, sourced from audited SEC filings via SimFin.

## What This Project Includes
- **Data Cleaning:** Raw financial statements restructured into clean, 
  analysis-ready tables with documented handling of missing values
- **Ratio Analysis:** Calculated profitability, growth, liquidity, leverage, 
  and return metrics using live Excel formulas
- **Interactive Dashboard:** 5 KPI cards and 7 charts summarizing 10 years 
  of financial performance

## Key Findings
- Revenue declined 33% from its FY2012 peak, while gross margin stayed 
  stable at 60–64% throughout the decade
- FY2017 net income crashed due to a one-time U.S. tax reform charge, 
  not an operational issue — confirmed by stable free cash flow that year
- Debt-to-Equity rose from 0.47x to 2.29x over the decade, funding 
  dividends and share buybacks, while liquidity remained stable

## Tools Used
- Microsoft Excel (data cleaning, formulas, PivotCharts, dashboard design)
